
# Secret Auction Program

A simple secret auction program written in Python. Participants submit bids privately and the program determines the winner without revealing other bids.

- **Language:** Python
- **Files:** `main.py`, `art.py`

**Usage:**

Run the program with Python (requires Python 3.7+):

```powershell
python main.py
```

**Flowchart (program):**

![Flowchart](Secret%20Auction%20Program.jpg)

**What it does:**

- Accepts bids from multiple users.
- Keeps bids private while collecting them.
- Announces the highest bidder and amount.
